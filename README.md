# 📖 Project Overview

This project develops an AI-powered decision support system that analyzes historical crude oil prices, engineers time-series features, trains multiple machine learning models, evaluates forecasting performance, and presents insights through an interactive Power BI dashboard.

---

# 🎯 Objectives

- Analyze Brent & WTI historical prices
- Compare nominal vs inflation-adjusted prices
- Forecast future oil prices
- Compare ML models
- Visualize insights in Power BI
- Support data-driven decision making
